# MONIEZI Raise V1.5 — Premium Investor Portal Builder

V1.5 refocuses MONIEZI Raise into one product: a premium investor portal builder for the business owner.

## Core outcome

The business owner fills a guided template and generates a polished investor-facing portal for their own company.

## Main changes

- Removed legacy dashboard/tracker-style app flow.
- Reduced the app to Build, Preview, and Export.
- Added breathable guided builder sections with clear explanatory bullets.
- Added premium investor portal templates:
  - Premium Investor Portal
  - Startup / SAFE Portal
  - Local Business Expansion Portal
- Added owner-upload support for:
  - company logo
  - hero image
  - agreement / SAFE / funding document
- Rebuilt the investor portal preview as a full-width, mobile-first, one-column page.
- Added investor-facing agreement download behavior.
- Added investor interest form area that prepares an email to the owner in exported static HTML.
- Added static investor portal HTML export.
- Added builder profile JSON export.
- Added private funding package text export.

## Mobile layout rule

The investor portal uses full-width sections with readable internal spacing. It avoids narrow nested frames, two-column layouts, and squeezed preview boxes.

## Safety wording

The portal uses careful language such as non-binding indication of interest, owner review, final documents, advisor-reviewed agreement, and direct payment-instruction verification.
