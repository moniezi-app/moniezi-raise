# MONIEZI Raise V1.5.5 — Premium Section Builder

This version focuses on premium builder presentation and breathing room.

## Changes

- Increased line spacing across headings, body copy, guidance text, labels, fields, and portal copy.
- Rebuilt builder section headers as premium dark gradient bands instead of plain white blocks.
- Converted field groups into distinct card-style builder areas so each major input is easier to see.
- Added stronger visual hierarchy for important sections such as business overview, investor thesis, why now, terms summary, agreement title, investor process, agreement summary, and risk review notice.
- Preserved section-level image and video upload support.
- Updated version to 1.5.5 and bumped the service-worker cache.

## Packaging

Source-only package. No node_modules. No dist. GitHub Pages workflow included.
