# MONIEZI Raise V1.2.1 — Dark-Mode White Blocks Hotfix

This build keeps the standalone MONIEZI Raise workflow and fixes the dark-mode selected-tab/action-button contrast problem.

## Fixed

- Removed the blank white selected tab blocks in dark mode.
- Fixed the bottom navigation selected item so it is no longer a white square with invisible text.
- Fixed the top navigation active tab.
- Updated action buttons to use blue high-contrast styling instead of white-on-white in dark mode:
  - Export Profile
  - Export CSV
  - Download Package
  - Next
- Updated the package/app version to `1.2.1`.
- Bumped the service-worker cache so Android/Chrome should pick up the new build.

## Included

- Raise Setup for company profile, founder contact, round terms, SAFE/private raise details, use of funds, acknowledgments, and wire-security notice.
- Company profile export/import so future MONIEZI Pro Finance builds can transfer company details into Raise.
- Six-step investor/signup flow.
- Downloadable investor package as a standalone HTML review file that can be opened in a browser and printed/saved to PDF.
- Investor submission tracker with status updates and CSV/JSON exports.
- Mobile bottom navigation focused only on Raise workflows.

## Product positioning

MONIEZI Pro Finance remains the owner/admin business finance and internal private raise tracker. MONIEZI Raise is the standalone visitor-facing page-app for private funding interest and package review.

## Compliance language

The app uses proposed pledge, reservation, and indication-of-interest wording. It does not claim to issue securities, complete a sale, or replace legal review, final documents, payment verification, or securities-law compliance.
