# MONIEZI Raise V1.5.2 — Premium Typography Polish

V1.5.2 updates the visual feel of the investor portal builder and generated portal.

Changes:

- Replaced the old Inter-only look with a modern Manrope UI font and Sora display-heading stack.
- Reduced heavy all-bold typography that made the app feel dated and blocky.
- Softened headings, labels, buttons, form text, guidance cards, preview sections, and exported portal typography.
- Updated the exported static investor portal HTML so the published page uses the same modern typography.
- Updated package version, app title, manifest id, and service-worker cache.

Packaging rules preserved:

- Source-only ZIP.
- No node_modules.
- No dist.
- GitHub Pages workflow included.
