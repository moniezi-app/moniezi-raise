# MONIEZI Raise V1.3.1 — Full Investor Portal Template

## Purpose

V1.3.1 upgrades MONIEZI Raise from a thin funding-page preview into a fuller investor/supporter portal builder.

The first template is an EMYRGEN-style Investor Portal structure that a buyer can customize for their own company. It is not a copy of EMYRGEN content; it uses the same serious investor-page logic: offer, why now, business model, proof, market opportunity, process, signup, package, and review notices.

## Key changes

- Added a third template: EMYRGEN-style Investor Portal.
- Expanded Build flow with fields for investment thesis, why-now story, business model, proof/readiness, market opportunity, and process steps.
- Rebuilt Preview into a longer investor-portal page rather than a small summary card.
- Updated static HTML export so exported pages match the richer portal preview.
- Updated package generator to include the investment case.
- Bumped app version and service-worker cache to V1.3.1.

## Safe positioning

MONIEZI Raise collects non-binding indications of interest and helps owners prepare a private funding page and package. It does not guarantee funding, issue securities, act as a crowdfunding portal, or replace legal/accounting review.
